# Spanish Preterite Stem and Spelling Changes

This document summarizes the rules for stem-changing and spelling-changing verbs in the Spanish Preterite tense.

---

### Rule 1: -AR and -ER Verbs Do Not Stem-Change

Verbs ending in -ar and -er that have a stem change in the Present tense **do not** change in the Preterite.

| Verb | Present Tense Change | Preterite Tense (No Change) |
|---|---|---|
| **pensar** (to think) | p**ie**nso | yo p**e**nsé |
| **contar** (to count) | c**ue**nto | yo c**o**nté |
| **entender** (to understand)| ent**ie**ndo | yo ent**e**ndí |
| **volver** (to return) | v**ue**lvo | yo v**o**lví |

---

### Rule 2: -IR Verbs Have Limited Stem Changes

Stem changes in the Preterite only occur in **-IR verbs**, and only in the **third-person singular (él, ella, usted)** and **third-person plural (ellos, ellas, ustedes)** forms.

#### Type 1: `e -> i`

This change affects verbs like `pedir`, `sentir`, `repetir`, `preferir`, `seguir`, `servir`.

**Example: `sentir` (to feel)**

| Person | Conjugation |
|---|---|
| yo | sentí |
| tú | sentiste |
| **él/ella/ud.** | **sintió** |
| nosotros | sentimos |
| vosotros | sentisteis |
| **ellos/ellas/uds.**| **sintieron**|

#### Type 2: `o -> u`

This change affects verbs like `dormir` and `morir`.

**Example: `dormir` (to sleep)**

| Person | Conjugation |
|---|---|
| yo | dormí |
| tú | dormiste |
| **él/ella/ud.** | **durmió** |
| nosotros | dormimos |
| vosotros | dormisteis |
| **ellos/ellas/uds.**| **durmieron**|

---

### Rule 3: Spelling Changes in the "Yo" Form

These changes happen in the first-person singular (`yo`) to maintain the original sound of the consonant.

*   **-car -> -qué**: `buscar` -> `busqué`, `tocar` -> `toqué`, `practicar` -> `practiqué`
*   **-gar -> -gué**: `llegar` -> `llegué`, `pagar` -> `pagué`, `jugar` -> `jugué`
*   **-zar -> -cé**: `empezar` -> `empecé`, `almorzar` -> `almorcé`, `organizar` -> `organicé`

---

### Rule 4: "Y" Verbs and Irregular Stems

#### Type 1: `i -> y` Spelling Change

For -ER and -IR verbs where the stem ends in a vowel (like `leer`, `caer`, `oír`, `creer`), the `i` of the ending becomes a `y` in the third-person singular and plural.

**Example: `leer` (to read)**

| Person | Conjugation |
|---|---|
| yo | leí |
| tú | leíste |
| **él/ella/ud.** | **leyó** |
| nosotros | leímos |
| vosotros | leísteis |
| **ellos/ellas/uds.**| **leyeron**|

#### Type 2: Verbs with Fully Irregular Stems

A group of common verbs have their own unique stems in the Preterite. These new stems are used for **all** persons. They also share a unique set of endings: `-e, -iste, -o, -imos, -isteis, -ieron / -eron`.

| Verb | Preterite Stem | Example (`yo`, `él`) |
|---|---|---|
| **tener** | `tuv-` | `tuve`, `tuvo` |
| **estar** | `estuv-` | `estuve`, `estuvo` |
| **poder** | `pud-` | `pude`, `pudo` |
| **poner** | `pus-` | `puse`, `puso` |
| **saber** | `sup-` | `supe`, `supo` |
| **querer** | `quis-` | `quise`, `quiso` |
| **hacer** | `hic-` / `hiz-` | `hice`, `hizo` |
| **venir** | `vin-` | `vine`, `vino` |
| **decir** | `dij-` | `dije`, `dijo` (*dijeron*) |
| **traer** | `traj-` | `traje`, `trajo` (*trajeron*) |
| **conducir** |`conduj-`| `conduje`, `condujo` (*condujeron*) |

*Note on "j-stem" verbs*: The third-person plural ending is `-eron` (e.g., `dijeron`), not `-ieron`.
